# NM Helper

## Requirements
- macOS
- Xcode (SwiftUI app, `lipo`, `ditto`, SDKs)
- Xcode Command Line Tools (`xcode-select --install`)

## Optional
- Homebrew

## Build Overview
- SwiftUI app lives in `NM_View/`
- Swift `nmcore` package lives in `nmcore/`

## Build
- Open `NM_View/NM_View.xcodeproj` in Xcode and Run.
